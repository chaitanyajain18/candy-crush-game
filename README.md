## Description
This is a browser-based puzzle game inspired by Candy Crush.
The player matches similar candies to score points and clear the board.

## Features
- Match-3 game logic
- Score tracking
- Dynamic grid
- Smooth animations
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Clone the repository
2. Open `index.html` in your browser
3. Match 3 or more candies of the same type
